# TaskManager
A task management application which keeps you in charge of your life! TaskManager makes it easy for you to track to-dos, manage task executions and keeps you reminded of what you need to get work done!

## Technologies used
1. Python/Flask
2. Postgres database
3. SqlAlchemy ORM
